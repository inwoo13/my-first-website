인터넷이 동작하기 위해서는 컴퓨터가 최소 2대가 필요.
요청하는 컴퓨터를 client컴퓨터 응답하는 컴퓨터를 server 컴퓨터라고 하기로 함.
web server -> 자신의 컴퓨터에 웹서버를 직접 깔아서 하는것. 어려움.
web hosting -> 대행업체에 맡기는 것. 쉬움.

인터넷에 연결된 컴퓨터 하나 하나를 호스트라고 함. 이런 컴퓨터를 빌려주는 사업을 호스팅이라 한다. 웹 호스팅을 github pages를 이용하여 할 수 있음.

github에서 repository를 생성
->new버튼 클릭
->public or private을 이용도에 따라서 설정
->create repository
->uploading an exisitng
->작업한 파일 업로드
->사이드 바에서 pages를 선택
->branch를 main or master 선택 and save
->자신이 만든 웹페이지 완성
